这是一个基于 Python Flask 的网页应用，用户可以上传评论数据（CSV 或 Excel 格式），通过调用 DeepSeek 的大语言模型（LLM）接口，自动提取以下信息：

情感倾向（Sentiment）：正面 / 中性 / 负面  
评论类型（Category）：例如产品质量、客户服务、配送等  
关键词主题（Key Themes）：简短短语提炼核心观点  

## 功能展示

1. 登录页面（用户名 Jacob，密码 2005）  
2. 文件上传页面 支持csv 文件
3. 处理完成后以表格形式展示分析结果
